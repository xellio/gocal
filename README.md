## gocal - a cli calendar presentation written in go

[Documentation](https://godoc.org/github.com/xellio/whois)

![gocal cli calendar](./gocal.png?raw=true "gocal cli calendar")

## Usage
```
calendar := gocal.Cal{}
calendar.Print()
```
